# framework-laptop-meta
Metapackage to install optimal utilities and tweaks for Framework laptops on Arch Linux.
Dependencies

Mandatory:

power-profiles-daemon
fprintd
brightnessctl
bluez, bluez-utils
fwupd
linux-firmware
i2c-tools

Optional:

amd-ucode / intel-ucode
via
thermald
tlp
iio-sensor-proxy
ectool-framework-git
